**Project Overview**

This project features an interactive Power BI dashboard designed to analyze business data and present key insights through clear and engaging visuals.

**Objectives**

Analyze structured business data to uncover trends and insights

Build an interactive and easy-to-understand Power BI dashboard

Present key KPIs using effective visual storytelling

Enable quick decision-making through filters and drill-downs

**Tools & Technologies**

Power BI Desktop

Power Query – Data cleaning and transformation

DAX – Calculated measures and KPIs

Excel / CSV – Source data (if applicable)

**Insights Generated**

Identification of high-performing and low-performing segments

Trend analysis over time

Comparison across categories/regions/metrics

Summary view for quick business understanding
